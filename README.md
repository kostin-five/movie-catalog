# 🎬 Каталог Фильмов (Movie Catalog)

Интерактивное веб-приложение для поиска и просмотра популярных фильмов, разработанное на React. Проект использует API The Movie Database (TMDB) для получения актуальных данных о кинематографе.

## ✨ Ключевые особенности

- **Бесконечная лента (Infinite Scroll):** Плавная подгрузка новых фильмов при прокрутке страницы вниз, реализованная с помощью нативного API `IntersectionObserver`.
- **Продвинутая работа с состоянием:** Использование кастомного хука `useFetching` для элегантной и переиспользуемой обработки состояний загрузки и ошибок при запросах к серверу.
- **Маршрутизация (Routing):** Навигация между главной страницей и детальными страницами фильмов с использованием современного API React Router.
- **Архитектура:** Разделение ответственности — вынесение сетевых запросов в отдельный сервисный класс `MovieService` и изоляция UI-компонентов.

## 🛠 Технологический стек

- **Frontend:** React 19
- **Сборщик:** Vite
- **Навигация:** React Router DOM 7
- **Сетевые запросы:** Axios
